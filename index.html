Recent Projects

Project 1

EDUCATION CERTIFICATES AND CREDENTIALS VAULT USING BLOCKCHAIN

Education Certificates and Credentials Vault Using Blockchain is a secure and decentralized platform for managing academic credentials. By leveraging blockchain technology, specifically Polygon, and storing certificates on IPFS, it ensures tamper-proof records and easy verification of qualifications. The system uses smart contracts to manage the issuance and retrieval of certificates, providing transparency and security. Users can access their credentials through a web interface integrated with Web3.js and MetaMask, ensuring privacy and control over personal data. This solution addresses the inefficiencies of traditional credential management, such as forgery, loss, and the manual verification process.

Project 2

HASHED PASSWORD CRACKER

Hashed Password Cracker is a tool designed to recover or crack hashed passwords by using various techniques like brute-force, dictionary attacks, or rainbow tables. It takes a hashed password as input and attempts to find the original plaintext password by generating possible combinations and comparing their hashes to the given one. The tool aims to highlight the vulnerabilities in weak hashing algorithms and passwords, emphasizing the importance of using strong encryption and complex passwords to secure sensitive data.

Project 3

BEACH FINDER: RECREATIONAL SUITABILITY LOCATOR

Beach Finder: Recreational Suitability Locator is a mobile application that aims to enhance the beach-going experience across India by providing users with real-time information on beach suitability. It assesses factors such as safety, water quality, weather conditions, and recreational facilities, allowing users to make informed decisions about which beaches to visit. The app features a user-friendly interface and interactive maps, helping beachgoers find ideal spots for relaxation and activities. By promoting safe and enjoyable beach experiences, this app contributes to better environmental awareness and public health.

Push-Up Counter is an IoT project designed to track and count push-ups in real-time using sensors.
